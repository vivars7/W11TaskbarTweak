# Windows 11 Taskbar Tweaker

## Introduction
:wrench: Using this tweak, you can move the Taskbar on Windows 11<p/>
**:warning: Warnings**
1. This tweak only supports for Windows11.
2. This tweak needs to restart "Windows Explorer" process.
